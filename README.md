# 🌌 Quantum Consortium Chain

> **量子安全联盟链** — BB84 QKD × PBFT Consensus

将 BB84 量子密钥分发协议与联盟链深度融合，构建抗量子计算攻击的安全区块链。

---

## 🎯 核心思想

传统区块链依赖 ECDSA/RSA 签名，在量子计算机面前形同虚设。本项目将 **BB84 QKD 生成的量子密钥** 注入联盟链的共识和签名流程，实现：

```
┌──────────────────────────────────────┐
│          量子安全联盟链               │
│                                      │
│   BB84 QKD ──→ 量子密钥 ──→ 签名     │
│                    ↓                 │
│              交易加密 + 区块签名      │
│                    ↓                 │
│              PBFT 共识出块            │
└──────────────────────────────────────┘
```

## 🔬 技术细节

### BB84 QKD 流程

| 步骤 | 操作 | 信道 |
|------|------|------|
| 1 | Alice 随机选基制备量子比特 | - |
| 2 | 量子信道传输 | 量子信道 |
| 3 | Bob 随机选基测量 | - |
| 4 | 经典信道基比对 | 经典信道 |
| 5 | QBER 窃听检测 | - |
| 6 | 纠错 + 隐私放大 | - |
| 7 | 生成最终安全密钥 | - |

### PBFT 共识

- **3f+1** 容错模型
- Pre-Prepare → Prepare → Commit 三阶段
- 支持动态节点加入

### 量子安全机制

- ✅ QKD 密钥签名代替 ECDSA
- ✅ 交易数据 QKD 加密
- ✅ 窃听自动检测 (QBER 阈值)
- ✅ 密钥轮换支持
